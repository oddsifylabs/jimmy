Getting Started with Jimmy

Jimmy makes it easy to check your sports bet results instantly. Here's everything you need to know:

Quick Check

Paste your bets and click "Check Results" to instantly see WIN/LOSS status

Export Data

Save your bets as JSON files for backup and record keeping

Import Bets

Upload AlexBet CSV, simple CSV, or JSON files

Multiple Formats

Handle spreads, moneylines, props, and totals

Bet Format

Jimmy accepts bets in this format:

Team Name Spread - MM/DD/YYYY

Examples:

  • Carolina Hurricanes -1.5 - 05/08/2026
  • Houston Astros ML - 05/08/2026
  • Toronto Raptors +8.5 - 04/29/2026
  • Over 45.5 - 05/11/2026

Import Options

Option 1: AlexBet Signal Pro CSV

Simply upload your exported CSV from AlexBet. Jimmy automatically parses the format.

Option 2: Simple CSV

Create a simple CSV with 3 columns:

Team,Spread,Date Carolina Hurricanes,-1.5,05/08/2026 Houston Astros,ML,05/08/2026 Toronto Raptors,+8.5,04/29/2026

Option 3: Jimmy JSON Export

Export bets from Jimmy and re-import them anytime. Perfect for backups and sharing.

Step-by-Step Guide

  • 1Paste your bets in the text area or import from a file
  • 2Click "Check Results" to scan for game outcomes
  • 3View WIN/LOSS badges for each bet
  • 4Export your results for record keeping

Supported Bet Types

  • Spreads: -1.5, +8.5, +2 (any decimal)
  • Moneylines: ML or Moneyline
  • Totals: Over 45.5, Under 200
  • Props: Any team/player name with spread
  • Parlays: Enter each leg separately

Pro Tips

  • Use Ctrl+Enter (or Cmd+Enter on Mac) for quick submission
  • Dates work with single digits (4/28/2026) or padded zeros (04/28/2026)
  • Import merges with existing bets - you can build a history over time
  • Export regularly to backup your bet records
  • Jimmy stores data locally - nothing is sent to servers

Frequently Asked Questions

Q: What if a bet isn't in the database?

A: Jimmy will show "PENDING" status. You can still export and track it.

Q: Can I import multiple files?

A: Yes! Import files one at a time and they'll all merge together.

Q: Is my data safe?

A: Completely! All data stays on your device. Nothing is uploaded anywhere.

Q: Can I use Jimmy offline?

A: Yes! Save the HTML file and use it completely offline.
